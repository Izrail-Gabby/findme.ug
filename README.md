# FindMe.ug  
**Your trusted fundi is just one click away**  

Uganda’s fastest-growing hyper-local service marketplace  
Starting in Kampala → Nationwide  

### Live Site (coming soon)
https://findme.ug

### Problem
Every day, thousands of Ugandans struggle to find reliable plumbers, carpenters, mechanics, salon professionals, etc.

### Solution
FindMe.ug connects customers with verified local workers instantly via WhatsApp & Call.

### Tech Stack
- Frontend: React  
- Backend: Node.js + Express  
- Database: MongoDB Atlas  
- Payments: Mobile Money (Flutterwave)  
- Notifications: WhatsApp + SMS (Africa’s Talking)
